# Materials Informatics Demo

Hello everyone! 👋

Welcome to **Materials Informatics Demo**, an open-source organization created to support the textbook series ***An Introduction to Materials Informatics*** by **Prof. Tong-Yi Zhang**.

Our mission is to make materials informatics more accessible through open-source code, datasets, tutorials, and educational resources. The repositories in this organization are intended to complement the books and help students, researchers, and educators better understand the concepts through hands-on examples.

We warmly welcome contributions from the community. If you find bugs, mistakes, or have suggestions for improving the code or educational materials, please feel free to open an issue or submit a pull request. Your feedback helps us continuously improve the quality of these resources.

Together, we hope to build an open, reliable, and practical learning platform for the materials informatics community.
